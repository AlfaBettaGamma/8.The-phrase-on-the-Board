
Фраза на доске

Учительница заставила Олега, забывшего домашку по алгебре, много-много раз написать на доске фразу "а голову ты не забыл?". Однако Олег придумал способ кодирования количества многократного повторения любой фразы в ней самой. Путём дублирования некоторых букв во фразе мы получаем некоторое обобщённое множество её представлений. Например, во фразе "аа голову ты не забыыл??" восемью способами может быть выделена фраза "а голову ты не забыл?".

Более формально, надо получить количество всех возможных последовательностей возрастающих индексов символов из обобщённой строки, которые дают искомую строку.

Например, есть строка "123" и есть её обобщённая форма "1102353".
С учётом того, что индексы начинаются с нуля, получим четыре варианта возможных последовательностей индексов:

0,3,4
0,3,6
1,3,4
1,3,6

Функция

int HowManyTimes(string s, string s_generic)

получает исходную строку s и её обобщённый вариант s_generic, и возвращает целое число -- количество "вхождений" s в s_generic по вышеуказанной схеме (количество возможных последовательностей индексов).
